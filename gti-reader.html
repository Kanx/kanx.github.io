GTi Reader is an Angular 1 plugin that can be used to import and display programmes from GTi in to a customisable component.

Default settings

To display a feed showing all open programmes, declare a programme-lister directive; simple!

Markup

<div programme-lister></div>

Output

Filtering

The programme-lister directive on its own is useful for displaying all programmes, but when you're dealing with a programme list that's 10 or more programmes large, you'll want to provide users with a means of refining the results.

To add filtering to the feed, add a new element with the a programme-filter directive. By default, there will be no filter controls; these must be added as additional attributes. The 4 control directives are:

  • textsearch - A free-text keyword search input
  • business - A select box displaying all available business areas
  • location - Another select box displaying all available locations
  • programme - And yet another select box dsplaying all programme types (Graduate/intern)

This example shows a component only using business and programme:

Markup

<div programme-lister>
	<div programme-filter business programme></div>
</div>

Output

Restricting programmes

It is possible to restrict the scope of programmes available (with or without filtering) by specifying attributes in the top-level programme-lister directive. If used in conjunction with programme-filter the available filter options will be pre-selected accordingly.

Note that if a value is mis-typed (e.g. location="Machezter" then no results will show). The following attributes can be used to filter results sets:

  • business
  • location
  • programme

Markup

<div programme-lister location="Manchester" programme="Internship">
	<div programme-filter freetext business programme location></div>
</div>

Output
